# UNIDEV — Trilha de Programação (Fundamentos)

Objetivo da trilha:

Desenvolver **lógica de programação, pensamento computacional e fundamentos de desenvolvimento**, utilizando **C como linguagem base**.

Carga sugerida: **8 encontros**

---

# Aula 1 — Introdução à Programação

Conteúdos:

- O que é programação
- O que é algoritmo
- Estrutura de um algoritmo
  - entrada
  - processamento
  - saída
- Fluxograma
- Introdução ao Portugol
- Primeiro algoritmo em Portugol
- Tradução do algoritmo para C
- Primeiro programa em C

---

# Aula 2 — Variáveis e Entrada de Dados

Conteúdos:

- Estrutura básica de um programa em C
- Tipos de variáveis
  - int
  - float
  - char
- Declaração de variáveis
- Entrada de dados com `scanf`
- Saída de dados com `printf`

Exercícios:

- ler nome e idade
- calcular idade futura
- converter metros para centímetros

---

# Aula 3 — Operadores e Cálculos

Conteúdos:

- Operadores matemáticos
  - +
  - -
  - *
  - /
  - %
- Expressões matemáticas
- Ordem das operações

Exercícios:

- calculadora simples
- cálculo de média
- cálculo de IMC

---

# Aula 4 — Estruturas Condicionais

Conteúdos:

- Estrutura `if`
- `if / else`
- operadores relacionais
  - >
  - <
  - >=
  - <=
  - ==
  - !=

Exercícios:

- número par ou ímpar
- aprovação de aluno
- maior de três números

---

# Aula 5 — Estruturas de Repetição

Conteúdos:

- loop `for`
- loop `while`
- controle de repetição

Exercícios:

- contar números
- tabuada
- soma de vários números
- jogo de adivinhação

---

# Aula 6 — Vetores (Arrays)

Conteúdos:

- conceito de vetor
- declaração de vetor
- acesso por índice
- percorrer vetor com loop

Exercícios:

- armazenar vários números
- calcular média de turma
- encontrar maior número

---

# Aula 7 — Funções

Conteúdos:

- conceito de função
- criação de funções
- parâmetros
- retorno de valores

Exercícios:

- função de soma
- função de média
- função de verificação par/impar

---

# Aula 8 — Projeto Final

Projeto:

Sistema de **Almoxarifado em C (via terminal)**.

Funcionalidades:

- cadastrar item
- listar itens
- editar item
- deletar item

Conceitos utilizados:

- structs
- vetores
- funções
- busca em lista
- menu interativo

---

# Resultado esperado

Ao final da trilha os alunos terão:

- base sólida em lógica de programação
- noção de estrutura de programas
- prática com estruturas de controle
- primeiro projeto funcional em C
