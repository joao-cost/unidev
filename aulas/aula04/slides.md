# Aula 4 — Slides de Condicionais 🚦

## 🎯 Estrutura `if` / `else`
```c
if (condição) {
    // Verdade
} else {
    // Falso
}
```

## 🏛️ Operadores Relacionais
*   `>`, `<`, `>=`, `<=`.
*   `==` (Igual).
*   `!=` (Diferente).

---

## 📜 Estruturas Aninhadas
```c
if (n >= 7) { ... }
else if (n >= 5) { ... }
else { ... }
```
