# Aula 8 — Slides do Projeto Final 🏁

## 🎯 Objetivo: Sistema de Almoxarifado em C (Terminal)
Sistema de controle de estoque funcional via terminal para consolidar todo o conhecimento.

## 🏛️ Funcionalidades do Projeto
Menu interativo com as seguintes opções:
1.  **Cadastrar Item:** Adição de novos produtos (Nome e Quantidade).
2.  **Listar Itens:** Mostrar tudo o que há no estoque.
3.  **Editar Item:** Alterar o nome ou a quantidade.
4.  **Deletar Item:** Exclusão de itens.
5.  **Encerrar Programa.**

---

## 📜 Conceitos Utilizados
*   **Structs**: Organização dos dados do item.
*   **Vetores**: Armazenar a lista de itens.
*   **Loops**: Manter o menu interativo rodando.
*   **Funções**: Cada funcionalidade em seu próprio bloco.
