# Aula 7 — Slides de Funções ⚙️

## 🎯 Conceito de Funções
Bloco de código independente e reutilizável.
Evita cópia de código e organiza o programa principal (`main`).

## 🏛️ Criação de Funções
1.  **Tipo de Retorno** (`int`, `float`, `void`).
2.  **Parâmetros** (Daddos de entrada).
3.  **Bloco de Código** e **`return`**.

```c
int somar(int a, int b) {
    return a + b;
}
```

---

## 📜 Chamada
Chamamos ela por dentro do `main` passando valores reais.
