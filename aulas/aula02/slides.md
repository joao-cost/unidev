# Aula 2 — Slides de Variáveis e Dados 🏗️

## 🎯 Estrutura Básica de um Programa C
```c
#include <stdio.h>
int main() {
    return 0;
}
```

## 🏛️ Tipos de Variáveis
*   **`int`**: Números inteiros.
*   **`float`**: Números com casas decimais (reais).
*   **`char`**: Caracteres simples.

---

## 📜 Declaração de Variáveis
`int idade;`
`float altura;`

---

## 📥 Entrada e Saída
*   **`printf`**: Saída de dados (mostrar na tela).
*   **`scanf`**: Entrada de dados (ler do teclado).

---

## 💡 Dica: Formatos
*   `%d`: Int.
*   `%f`: Float.
*   `%c`: Char.
