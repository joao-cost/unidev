# Aula 5 — Slides de Repetição 🔁

## 🎯 Loops
Repetir blocos de código.

## 🏛️ `for`
Quando sabemos o limite.
```c
for (int i = 0; i < 10; i++) { ... }
```

## 📜 `while`
Enquanto for verdade.
```c
while (condição) { ... }
```

---

## 🚫 Loop Infinito
Cuidado! A condição precisa ficar falsa em algum momento.
