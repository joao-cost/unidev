## 📖 Sobre a Célula
A **UNIDEV** é uma célula de estudos voltada ao aprendizado de **programação, lógica e fundamentos de desenvolvimento**, formando uma comunidade sólida de alunos que aprendem, praticam e evoluem juntos.
